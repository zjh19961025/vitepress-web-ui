# select下拉状态

### 用法
:::demo
selectLoadMore/selectLoadMore
:::


### 属性（props）

| 属性名  | 说明                          | 类型    | 默认值 |
| ------- | ----------------------------- | ------- | ------ |
| loading | 弹窗loading状态               | boolean | -      |
| length  | 输入框显示options数据的length | number  | -      |
| leave   | 剩余的条数                    | number  | -      |
