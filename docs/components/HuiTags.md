### 属性

| 属性名   | 说明                                           | 类型                                                      | 默认值  |
| -------- | ---------------------------------------------- | --------------------------------------------------------- | ------- |
| text     | 标签显示的文本                                 | string                                                    | -       |
| tagType  | 标签的类型                                     | 'primary' \| 'success' \| 'warning' \| 'danger' \| 'info' | primary |
| tagClass | 当类型的颜色不满足的时候可以传入标签的背景颜色 | string                                                    | -       |
